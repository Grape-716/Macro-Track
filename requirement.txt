These are required libraries and modules in order to run application. Keep in mind application will not run without These.

You do not need to run any of these commands python program will handle all of it this a backup for manual install. If you face problems.

==========================================================================================

development library: customtkinter
to install customtkinter: Run in terminal "pip install customtkinter"

Ai module : gemeni 2.0
to install gemeni: Run in terminal "pip install google-generativeai"

image use: pillow 
to install pillow: Run in terminal "pip install pillow"

further more: cv2
to install cv2: Run in terminal "pip install opencv-python"


pip install google-cloud-firestore


pip install --upgrade firebase-admin


installing genai: pip install --upgrade google-genai
